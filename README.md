# Goal-Tracker
How to run
cd goal-tracker-project
npm install
npm run dev
A simple Goal Tracker application built with React, Vite, and Material UI.

## Features
- Create goals
- Edit goals
- Delete goals
- Add progress
- XP and streak system
- Archive completed goals
- Pause and resume goals
- Multi-language support (English, Persian, Pashto)
- RTL/LTR support
- Dark and Light mode
- Responsive design

-  Technologies

- React
- Vite
- Material UI
- React Router DOM
- Context API
- LocalStorage
